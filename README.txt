HOW TO RUN PROJECT:

REQUIREMENTS:
VS CODE (OR ANY OTHER IDE FOR PYTHON)
PYTHON EXTENTION
CONDA DATABASE CSV FILES PRESENT IN THE SAME FOLDER AS CODE

STEPS:
-LOAD THE WHOLE FOLDER INTO THE IDE;
-OPEN main.py;
-CLICK ON RUN;
-IF GETTING ERROR OF TYPE: "no module named x found" WRITE IN THE TERMINAL "pip install x" AND TRY AGAIN;
-ACCURACY RESULTS WILL BE WRITTEN IN THE TERMINAL;
-A CLASSIFICATION PREDICTION CSV WILL BE GENERARTED IN THE SAME FOLDER AS THE CODE AND DATABASE.
